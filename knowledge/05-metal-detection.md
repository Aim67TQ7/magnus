# Metal Detection Technical Reference

## Overview

Industrial metal detectors detect ALL metals — ferrous, non-ferrous (aluminum, copper, brass), and stainless steel — in product streams. Unlike magnetic separators which only capture ferrous materials, metal detectors identify and reject any metallic contamination. They are essential for food safety (HACCP/CCP), pharmaceutical compliance, and product quality verification.

## Equipment Types

### Conveyor Tunnel Metal Detectors
- Rectangular aperture surrounding a conveyor belt
- Detects all metal types passing through tunnel
- Automatic reject capability (air blast, pusher, diverter)
- Best for: packaged products, belt-conveyed bulk materials, inline inspection

### Free-Fall / Gravity Metal Detectors
- Circular aperture in vertical chute
- Material falls through detection zone
- Reject via diverter gate or air blast
- Highest sensitivity (smallest aperture = best detection)
- Best for: powders, granules, pellets in gravity flow

### Pipeline Metal Detectors
- Circular aperture around pipeline/pneumatic line
- Detects metal in pressurized or pneumatic conveying systems
- Best for: liquids, slurries, pneumatic conveying (dilute and dense phase)

## When to Use Metal Detection vs. Magnetic Separation

### Metal Detection Required When:
- Non-ferrous contamination possible (aluminum, brass, copper)
- Stainless steel detection needed (small fragments magnets miss)
- Verification/documentation after magnetic separator
- HACCP Critical Control Point compliance
- Automatic reject capability required
- Product liability exposure (pharma, medical, infant formula)
- Previous contamination incidents requiring proof of inspection

### Magnetic Separation Sufficient When:
- Only ferrous contamination present
- Equipment protection is primary goal (not product purity documentation)
- Cost optimization — magnets are simpler and cheaper
- No regulatory requirement for all-metal detection

### Best Practice: Combination Approach
**Magnetic separator FIRST → Metal detector SECOND**
1. Magnet removes bulk ferrous (reduces detector false triggers)
2. Detector catches remaining ferrous + all non-ferrous + stainless
3. Detector provides documentation and rejection capability
4. Standard HACCP/FSMA approach for food safety

## Maintenance & Troubleshooting

### Daily Checks
1. Function test of reject mechanism
2. Visual inspection for damage
3. Verify protective covers fitted and undamaged
4. Check for error messages

### Weekly Maintenance
1. Clean all product-contact surfaces
2. Verify installation supports secure
3. Check all cables and connections
4. Inspect light barriers and sensors
5. Clean display screen

### Monthly Maintenance
1. Check wire connectors for damage
2. Verify rotary encoder function (if installed)
3. Test separation device functionality
4. Test sensitivity with standardized test bodies
5. Verify complete production line operation

### Common Troubleshooting

**False Detection:**
1. Check for peripheral interference — switch off nearby machines
2. Look for intermittent metal-to-metal connections near detector
3. Perform "Learn Product" procedure for conductive products
4. Increase detection threshold to reduce sensitivity

**No Detection:**
1. Reduce detection threshold to increase sensitivity
2. Verify detector is in active mode (not deactivated)
3. Check power supply and connections
4. Examine transmitter and receiver cables for damage

**Separation Mechanism Failures:**
1. Check separation device function
2. Verify sensors/initiators and cables
3. Ensure adequate air pressure (pneumatic systems)
4. Test using "Test Reject Unit" function

## Sensitivity Optimization
1. Convey a metal part that should just be detected
2. If detected: gradually increase detection level until part is missed
3. Then reduce by 10% — this is your operating threshold
4. Document settings for each product profile

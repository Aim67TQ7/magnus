## Fail-Safe Rules (Always Check)

These critical warnings appear in F09 but should be surfaced whenever relevant:

| Condition | Warning | Source |
|-----------|---------|--------|
| Burden depth >6" + permanent plate magnet | FAIL — magnet cannot reach. Use electromagnet or head pulley | F09, F01 |
| Plastic film + standard gravity grate | FAIL — static clogging. Use drag slide | F09, F03 |
| Suspended magnet near steel structure | WARNING — structure will magnetize. 18-24" non-ferrous clearance required | F09, F02 |
| Gravity grate + bridging/sticky powder | FAIL — material bypasses. Use TurboGrate (rotary) | F09, F03 |
| Eddy current without prior ferrous removal | FAIL — ferrous damages ECS rotor | F09, F06 |
| Standard NdFeB at >176°F | FAIL — permanent demagnetization. Use high-temp NdFeB or SmCo | F09, F10 |
| Feeder tray near drum is mild steel | WARNING — tray will magnetize. Must be 300-series stainless | F04, F06 |

# Inline, Pneumatic & Liquid Separator Technical Reference

## Overview

Inline magnets, bullet magnets, and liquid traps serve pneumatic conveying lines, pressurized pipelines, and liquid/slurry applications where standard gravity or belt-mounted magnets cannot be used.

## Equipment Types

### Inline Cartridge / Bullet Magnets
- Aerodynamic nose cone design minimizes pressure drop
- Installed directly in pneumatic conveying lines
- Magnetic element captures ferrous as material flows around it
- Best for: dilute and dense phase pneumatic conveying of powders, pellets, granules

### Liquid Pressure Traps (Pipeline Separators)
- Installed in pressurized liquid pipelines
- Magnetic tubes or bars capture ferrous from liquids, slurries, sauces
- Sanitary versions for food/dairy (CIP-capable)
- Best for: viscous/sanitary (dairy, sauce, oil), liquid processing

### Wet Drum Separators
- Operate in slurry/water environment
- Heavy mineral slurry processing
- Best for: mining beneficiation, heavy media recovery

## Selection by Conveyance Mode

| Conveyance | Equipment | Key Consideration |
|------------|-----------|-------------------|
| Pneumatic (dilute phase) | Inline cartridge / bullet magnet | Must specify aerodynamic nose to minimize pressure drop |
| Pneumatic (dense phase) | Low pressure-drop inline magnet | Pressure drop is critical constraint |
| Liquid (viscous/sanitary) | Liquid pressure trap | Cleanability, CIP capability, sanitary construction |
| Liquid (heavy mineral slurry) | Wet drum separator | Mining-grade, high throughput |
| Screw/auger discharge | In-line housing magnet or grate at discharge | Position at material exit point |

## Critical Design Constraints

### Pneumatic Systems
- **Velocity too high** = primary failure mode — metal passes through field before capture
- Aerodynamic nose cone is mandatory to prevent material buildup and pressure loss
- Pressure drop specification is critical — excessive drop disrupts conveying system
- Consider both positive and negative pressure systems

### Liquid Systems
- **Cleanability** is primary concern — especially food/dairy/pharma
- **Pressure drop** — excessive restriction affects pumping and flow rates
- CIP (Clean-In-Place) capability eliminates manual disassembly
- Specify pipe diameter, flow rate (GPM), and viscosity for proper sizing

## Sizing Inputs

### Pneumatic Applications
| Parameter | Needed For |
|-----------|-----------|
| Pipe diameter (inches) | Inline magnet size selection |
| Flow rate (lbs/hr) | Capacity verification |
| Pressure type (positive/negative) | Design configuration |
| Material type | Magnetic circuit selection |

### Liquid Applications
| Parameter | Needed For |
|-----------|-----------|
| Pipe diameter (inches) | Trap size selection |
| Flow rate (GPM) | Capacity and pressure drop |
| Viscosity (water-like / moderate / thick) | Tube spacing and cleanability |
| Temperature | Magnet grade selection |
| Sanitary requirements | Construction material and certification |

## Industry Applications

### Plastics / Extrusion
- Inline cartridge in pneumatic pellet conveying lines
- Protects extruder screws and barrels from ferrous contamination
- Position in conveying line before feed hopper
- High-intensity rare earth for fine particle capture

### Food Processing
- Liquid traps in sauce, dairy, and beverage lines
- Sanitary 304/316 SS with CIP capability
- Must maintain food-grade certification
- Documentation for HACCP/FSMA compliance

### Chemical Processing
- Pipeline separators for chemical slurries
- Corrosion-resistant construction required
- Temperature specifications critical (process heat)

### Mining
- Wet drum separators for mineral beneficiation
- Multiple stages for increasing purity (LIMS → MIMS → HIMS)
- High throughput, heavy-duty construction
